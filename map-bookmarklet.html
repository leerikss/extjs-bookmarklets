Map Bookmarklet (Beta)

Installation

Map  <<-- Drag this link into your bookmarks bar, or right click and Add to Favorites

Usage

Select an addresse text on any page, and click the "Map" bookmark link. The bookmarklet pops up a floating resizable drag & drop window, and attempts to show the map position of the selected addresse using Google Maps.

Requirements

A valid browser; newest version of Chrome, Firefox, Opera, Safari, or IE 7+

Known issues

  • The Bookmarklet doesn't work on pages using framesets
  • Depending on the CSS used on various pages, the bookmarklet might look a bit strange
  • Depending on the JavaScript used on various pages, the bookmarklet might not work as expected, or not work at all (especially pages using ExtJS)
  • Text selection might not work on all pages
  • The bookmarklet loads dynamically quite an amount of resources (650 kB) from various locations. If some of the resources fail to load, the bookmarklet might look strange, or it may not work at all
  • IE: On some pages, won't work at all
